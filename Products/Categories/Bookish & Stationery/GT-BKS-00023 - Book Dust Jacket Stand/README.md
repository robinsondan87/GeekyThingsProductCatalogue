# Book Dust Jacket Stand

## Short Description
Honeycomb dust jacket display stand for showing off both a hardback and its dust jacket or book sleeve.

## Long Description
Display your books like the collector you are. This dust jacket display stand is designed to show the naked hardback alongside its matching dust jacket or book sleeve. The honeycomb cut-out design makes it a statement piece for any bookshelf while keeping the focus on your book.

## Features / Bullet Points

- Fits most standard hardbacks (custom sizing available).
- Displays dust jackets or book sleeves upright alongside the book.
