# AirTag Wallet Holder

SKU: GT-TCH-00011

## Short Description
Slim, credit-card style AirTag holder designed to sit inside a wallet or purse while keeping the tracker protected.

## Long Description
This low-profile AirTag holder is designed to slip into a wallet, purse, or handbag without adding bulk. It keeps your Apple AirTag secure and includes a silicone cover for added protection. Choose between the standard holder or a screw-back option depending on how secure you want the fit.

## Features / Bullet Points
- Slim profile (about 3.5 mm thick) for everyday wallets and purses.
- Holds an Apple AirTag securely; optional screw-back style.
- Includes a silicone cover to protect the AirTag.
- Lightweight, rigid 3D-printed plastic construction.
- AirTag not included.

## Variations / Sizes
Mounting style: Standard or screw-back.
