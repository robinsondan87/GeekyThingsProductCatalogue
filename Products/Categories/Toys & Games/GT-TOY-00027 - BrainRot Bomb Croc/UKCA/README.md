UKCA evidence checklist (per product)

Place files in these folders:
- Evidence/Drop_Test: drop test videos, notes, pass/fail, test conditions
- Evidence/Photos: product photos used for compliance records
- Evidence/Material_Certs: filament/material certificates (e.g., SunLu PLA 2.0+)
- Declarations: supplier declarations, UKCA declarations, conformity docs
- Labels: label artwork, placement photos, packaging labels
- Risk_Assessment: risk assessment document and any mitigations

Notes:
- Include version/size/color variants if applicable.
- Update ukca_status.csv when evidence is complete.
