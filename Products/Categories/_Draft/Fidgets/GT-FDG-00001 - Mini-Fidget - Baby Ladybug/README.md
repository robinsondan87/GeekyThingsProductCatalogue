# Mini-Fidget - Baby Ladybug

SKU: GT-FDG-00001

## Short Description
Mini-Fidget baby ladybug model with multicolor 3MF files, designed to print in parts without supports.

## Long Description
This baby ladybug model is designed with separated parts so it can be printed without supports. The creator notes the wings were a fun challenge to design, and the model can be scaled (example printed at 150%).

## Features / Bullet Points
- Multicolor 3MF files.
- Prints in parts without supports.
- Keychain option included.

## Variations / Sizes
Example print shown at 150% scale (creator note).

## Colours
Dark Blue, Red, Black (example filament colours used).

## Materials
PLA (eSun PLA Pro: Dark Blue, Red)
PLA (eSun ePLA Matte: Black)

## Print Settings
Wall line count: 3
Infill: 20%

## Assembly
Print in parts; assemble after printing.
